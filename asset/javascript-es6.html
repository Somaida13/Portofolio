Teori JavaScript Modern (ES6+)

Ringkasan fitur-fitur penting ES6 ke atas (ES6+), penjelasan singkat, dan contoh kode yang mudah dipahami.

ðŸ”° Apa itu JavaScript ES6+?

ES6 (ECMAScript 2015) adalah versi standar JavaScript modern yang memperkenalkan fitur-fitur baru untuk membuat kode lebih efisien, modular, dan mudah dibaca. Istilah ES6+ mencakup ES6 dan rilis selanjutnya.

Tujuan pembaruan: meningkatkan struktur kode, dukungan modular & asynchronous, dan mempermudah pengembangan aplikasi skala besar.

ðŸ§© Fitur-fitur Utama ES6+

1. let & const (Deklarasi variabel modern)

KeywordSifatKeterangan
varFunction scopeBisa redeclare & reassign (lama)
letBlock scopeBisa reassign
constBlock scopeTidak bisa reassign (konstan)
// contoh
let age = 21;
age = 22; // boleh

const pi = 3.14;
pi = 3.15; // error (tidak boleh diubah)

2. Arrow Function (Fungsi ringkas)

// versi lama
function greet(name) {
  return `Hello ${name}`;
}

// ES6 (arrow function)
const greet = name => `Hello ${name}`;

Arrow function lebih singkat dan tidak mengikat nilai this seperti function biasa.

3. Template Literals

const name = "Raka";
console.log(`Hai nama saya ${name}`);

4. Default Parameter

function sayHello(name = "Guest") {
  console.log(`Hello ${name}`);
}
sayHello(); // Hello Guest

5. Destructuring (Unpacking data)

// Array
const colors = ["red","green","blue"];
const [a,b] = colors; // a = "red", b = "green"

// Object
const user = { name: "Alya", age: 19 };
const { name, age } = user;

6. Spread & Rest Operator (...)

// Spread (menggabung / menyalin)
const arr1 = [1,2];
const arr2 = [...arr1, 3, 4]; // [1,2,3,4]

// Rest (mengumpulkan parameter)
function sum(...nums) {
  return nums.reduce((a,b) => a + b, 0);
}
console.log(sum(1,2,3,4)); // 10

7. Modules (import & export)

// export.js
export const name = "JavaScript";

// import.js
import { name } from "./export.js";

8. Class (Gaya OOP)

class Person {
  constructor(name) {
    this.name = name;
  }
  intro() {
    console.log(`Hello, I'm ${this.name}`);
  }
}
const p1 = new Person("Nia");
p1.intro();

9. Promise & Async/Await (Asynchronous modern)

// Promise
const getData = new Promise(resolve => {
  setTimeout(() => resolve("Data Loaded"), 1000);
});
getData.then(result => console.log(result));

// Async / Await
async function fetchData() {
  const data = await getData;
  console.log(data);
}
fetchData();

ðŸ“Œ Kesimpulan

ES6+ membuat JavaScript lebih modern, efisien, dan cocok untuk aplikasi web saat ini. Konsep inti yang penting dipelajari antara lain: let/const, arrow function, destructuring, modules, dan async/await. Pemahaman ES6+ sangat membantu ketika menggunakan framework modern seperti React, Vue, atau Angular.
