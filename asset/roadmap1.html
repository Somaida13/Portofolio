Full-Stack Developer adalah pengembang yang mampu bekerja baik di sisi **Frontend** (apa yang dilihat pengguna) maupun **Backend** (logika server dan database).

Frontend adalah sisi klien (browser). Fokusnya adalah pada tata letak, desain, dan interaksi visual pengguna.

  • **HTML:** Kerangka struktural konten.
  • **CSS:** Untuk styling dan estetika (warna, tata letak).
  • **JavaScript:** Untuk menambahkan interaktivitas dinamis (tombol, animasi).

Tanpa Frontend, tidak ada yang dapat dilihat atau diklik oleh pengguna akhir.

Backend berjalan di server dan bertanggung jawab atas semua logika di balik layar.

  • **Tugas Utama:** Memproses permintaan, mengautentikasi pengguna (login), dan menjalankan logika bisnis.
  • **Bahasa Populer:** Python (Django/Flask), JavaScript (Node.js/Express), PHP (Laravel).
  • **Hasil:** Menyediakan data yang terstruktur (seringkali dalam format JSON) melalui **API** ke Frontend.

Ini adalah bagian yang menentukan bagaimana data diproses dan disimpan dengan aman.

Database adalah tempat aplikasi menyimpan semua data secara permanen, seperti informasi pengguna, produk, atau transaksi.

  • **Relasional (SQL):** Data disimpan dalam tabel yang saling terhubung. Contoh: MySQL, PostgreSQL. Cocok untuk data terstruktur.
  • **Non-Relasional (NoSQL):** Data disimpan dalam format fleksibel (dokumen/key-value). Contoh: MongoDB, Redis. Cocok untuk data yang berubah cepat.

Backend bertanggung jawab untuk berkomunikasi dengan Database, menggunakan bahasa query khusus seperti SQL.
