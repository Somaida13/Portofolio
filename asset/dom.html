üìò Teori DOM (Document Object Model)

üîé Apa itu DOM?

DOM (Document Object Model) adalah representasi struktur dokumen HTML sebagai objek yang dapat diakses dan dimodifikasi menggunakan JavaScript.

üå≥ Struktur DOM sebagai Pohon

  • document sebagai root
  • Node merepresentasikan elemen, teks, komentar, dll.
  • Elemen dapat memiliki child (anak) sehingga membentuk tree.

üß≠ Node vs Element

Node mencakup semua entitas dalam DOM, sedangkan Element hanya node yang merupakan elemen HTML.

üõ† Cara Mengakses Elemen

document.getElementsByClassName("class")
document.querySelector("selector")
document.querySelectorAll("selector")

üß≠ Contoh Todo List Sederhana

<ul id="list"></ul>
<input id="txt">
<button id="add">Add</button>

<script>
  const list = document.getElementById('list');
  const txt = document.getElementById('txt');
  document.getElementById('add').addEventListener('click', () => {
    const li = document.createElement('li');
    li.textContent = txt.value;
    li.addEventListener('click', () => li.remove());
    list.appendChild(li);
    txt.value = '';
  });
</script>
