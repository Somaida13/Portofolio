HTML5 adalah versi terbaru dari HyperText Markup Language (HTML), bahasa markup standar yang digunakan untuk membuat dan menyusun konten di World Wide Web (WWW). Sebagai Calon Progremer, memahami HTML5 sangat penting karena ini adalah fondasi dari semua pengembangan front-end web modern.

1. Apa Itu HTML5?

HTML5 bukan sekadar pembaruan kecil; ini adalah revisi besar yang memperkenalkan banyak fitur baru, elemen, dan API (Application Programming Interfaces) yang dirancang untuk memungkinkan pengembangan aplikasi web yang lebih kaya dan interaktif.

  • HTML: Berfungsi untuk menyusun struktur dan konten dari sebuah halaman web (teks, gambar, tautan, dll.).
  • CSS (Cascading Style Sheets): Berfungsi untuk menentukan tampilan (warna, font, layout, dll.).
  • JavaScript: Berfungsi untuk menambahkan interaktivitas dan perilaku dinamis.

HTML5 bertujuan untuk mengurangi kebutuhan akan plugin pihak ketiga (seperti Flash) dengan menyediakan fungsionalitas media dan grafik secara native di browser.

2. Struktur Dasar Dokumen HTML5

Setiap dokumen HTML5 harus diawali dengan deklarasi doctype yang sederhana:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman Saya</title>
    <!-- Di sini tempat menautkan CSS dan JavaScript -->
</head>
<body>
    <!-- Konten yang terlihat oleh pengguna ada di sini -->
</body>
</html>
  • <!DOCTYPE html>: Deklarasi ini mendefinisikan dokumen sebagai HTML5. Ini harus selalu ada di baris pertama.
  • <html lang="id">: Elemen root dari halaman. Atribut lang="id" membantu browser dan mesin pencari mengetahui bahasa konten.
  • <head>: Berisi metadata tentang dokumen (informasi yang tidak ditampilkan langsung di halaman).
  • <body>: Berisi semua konten yang akan ditampilkan kepada pengguna.

3. Elemen Semantik Baru di HTML5

Salah satu pembaruan terbesar adalah pengenalan elemen semantik. Elemen semantik memberikan arti pada kontennya, membantu browser, search engine, dan pengembang lain memahami struktur halaman.

Elemen HTML5 Fungsi (Semantik) Analog Struktur Tradisional
<header> Konten pengantar, biasanya berisi logo dan heading situs. <div> dengan id="header"
<nav> Berisi tautan navigasi utama situs. <div> dengan id="nav"
<main> Konten utama dan unik dari dokumen. <div> dengan id="main-content"
<article> Konten yang berdiri sendiri (misalnya, postingan blog, berita). <div> dengan class="post"
<section> Bagian tematik yang mengelompokkan konten. <div> dengan class="section"
<aside> Konten yang kurang terkait, sering digunakan untuk sidebar. <div> dengan id="sidebar"
<footer> Konten penutup (informasi hak cipta, kontak, dll.). <div> dengan id="footer"

4. Elemen Media dan Grafik

HTML5 memungkinkan integrasi media tanpa plugin eksternal:

  • <video>: Untuk menanamkan video. 
    <video width="320" height="240" controls>
  <source src="video-kuliah.mp4" type="video/mp4">
  Browser Anda tidak mendukung tag video.
</video>
  • <audio>: Untuk menanamkan audio. 
    <audio controls>
  <source src="musik-intro.mp3" type="audio/mp3">
  Browser Anda tidak mendukung tag audio.
</audio>
  • <canvas>: Menyediakan area grafis di mana gambar dapat digambar menggunakan JavaScript.
  • <svg>: Untuk grafik vektor yang dapat diskalakan (*Scalable Vector Graphics*).

5. Peningkatan pada Form

HTML5 memperkenalkan tipe input baru untuk form yang meningkatkan pengalaman pengguna dan validasi data secara otomatis oleh browser.

Tipe Input Baru Deskripsi
type="email" Memastikan input berbentuk format email yang benar.
type="url" Memastikan input berbentuk URL.
type="number" Hanya menerima angka.
type="date" Menampilkan pemilih tanggal (*date picker*).
type="range" Menampilkan *slider* untuk memilih nilai dalam rentang tertentu.
required Atribut yang memastikan *field* tidak boleh kosong.

üë®‚Äçüíª Contoh dan Tutorial Singkat: Membuat Struktur Web Semantik

Berikut adalah kerangka sederhana dari sebuah halaman berita menggunakan elemen semantik HTML5.

Langkah 1 & 2: Struktur Dasar dan Penerapan Elemen Semantik

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Berita Teknologi Terbaru</title>
</head>
<body>
    <!-- 1. Header untuk Judul dan Logo Situs -->
    <header>
        <h1>SI News Center</h1>
        <p>Berita & Informasi Sistem Informasi Terkini</p>
    </header>

    <!-- 2. Navigasi Utama -->
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Programming</a></li>
            <li><a href="#">Database</a></li>
        </ul>
    </nav>

    <!-- 3. Konten Utama -->
    <main>
        <!-- Konten Berdiri Sendiri (Artikel) -->
        <article>
            <h2>Judul Artikel: Dampak AI pada Analisis Data</h2>
            <section>
                <h3>Pengantar</h3>
                <p>Kecerdasan buatan telah merevolusi cara kita memproses dan menganalisis data...</p>
            </section>
            <section>
                <h3>Kesimpulan</h3>
                <p>Penting bagi mahasiswa SI untuk menguasai alat-alat AI baru ini.</p>
            </section>
        </article>
    </main>

    <!-- 4. Sidebar/Konten Samping -->
    <aside>
        <h3>Artikel Populer</h3>
        <ul>
            <li><a href="#">Web Semantik</a></li>
            <li><a href="#">Deep Learning Basics</a></li>
        </ul>
    </aside>

    <!-- 5. Footer untuk Hak Cipta dan Info Kontak -->
    <footer>
        <p>&copy; 2025 SI News Center. Dibuat dengan HTML5.</p>
    </footer>
</body>
</html>

Hasil: Struktur dokumen Anda sudah sangat jelas dan terorganisir berkat elemen-elemen semantik HTML5.
