Figma dan Contohnya (Bahasa Mudah Dimengerti)

1. Apa Itu Figma?

Figma adalah alat desain berbasis web yang digunakan untuk membuat UI, UX, prototype, dan wireframe. Karena berbasis online, Figma bisa digunakan langsung dari browser tanpa instal aplikasi.

2. Mengapa Banyak Orang Menggunakan Figma?

  • Bisa kolaborasi real-time seperti Google Docs
  • Ringan karena berbasis web
  • Mudah melakukan export (PNG, JPG, SVG, PDF)
  • Banyak plugin gratis untuk icon, ilustrasi, warna, dsb.

3. Fitur Utama Figma

1) Frame

2) Components

Elemen reusable seperti tombol, card, atau navbar. Jika component induk berubah, semua salinannya ikut berubah.

3) Auto Layout

Fitur untuk menyusun elemen secara otomatis, mirip flexbox di CSS. Memudahkan membuat tombol atau card yang responsif.

4) Prototype

Menghubungkan halaman agar desain bisa diklik seperti aplikasi nyata. Cocok untuk presentasi ke user.

4. Alur Kerja Mendesain di Figma

  1. Menentukan ukuran frame
  2. Membuat wireframe (hitam-putih)
  3. Menambahkan warna, icon, typography
  4. Membuat prototype interaktif

5. Contoh Penggunaan Figma

Contoh 1: Halaman Login

Wireframe: input email, password, tombol login.

UI Final: warna, icon, logo.

Contoh 2: Aplikasi Food Delivery

Elemen: header, search bar, carousel promo, card makanan.

Prototype: klik produk ‚Üí masuk ke halaman detail.

Contoh 3: Dashboard Admin

Menggunakan auto layout untuk sidebar, tabel, dan grafik agar rapi dan presisi.

6. Kesimpulan

Figma adalah alat desain modern yang mudah digunakan, mendukung kolaborasi real-time, dan memiliki fitur lengkap untuk membuat UI/UX profesional.
