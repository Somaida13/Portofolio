Teori UI/UX (Bahasa Mudah + Contoh Praktis)

1. Apa Itu UI (User Interface)?

UI adalah tampilan visual yang dilihat pengguna, seperti warna, tombol, ikon, layout, dan font.

Contoh UI yang Baik

  • Tombol warna kontras
  • Font jelas
  • Layout rapi

Contoh UI Buruk

  • Warna terlalu mencolok
  • Font kecil
  • Elemen berantakan

2. Apa Itu UX (User Experience)?

UX adalah pengalaman pengguna saat memakai website atau aplikasi, termasuk kenyamanan dan kemudahan.

Contoh UX yang Baik

  • Proses mudah
  • Loading cepat
  • Error jelas

Contoh UX Buruk

  • Form terlalu banyak
  • Menu membingungkan
  • Error tanpa penjelasan

3. Perbedaan UI & UX

UIUX
Tampilan visualPengalaman
Warna, ikon, layoutAlur, kenyamanan
Bagian yang dilihatBagian yang dirasa

4. Prinsip Dasar UI

  • Konsistensi
  • Font mudah dibaca
  • Kontras warna
  • Hirarki visual

5. Prinsip Dasar UX

  • Sederhana
  • Mudah dipahami
  • Cepat
  • Minim error
  • Ada feedback

6. Contoh UI/UX pada Halaman Login

Contoh UI

<input type="email" placeholder="Masukkan email" /> <input type="password" placeholder="Masukkan password" /> <button>Login</button>

UX baik: tombol jelas, form sedikit, error mudah dimengerti.

7. Contoh UI/UX Buruk

  • Form 10 input
  • Tombol kecil dan warna tidak terlihat
  • Loading lama

Kesimpulan

UI = tampilan. UX = pengalaman. Keduanya harus bekerja bersama agar website enak dilihat dan digunakan.
