Teori CSS3: Flexbox dan Grid

CSS3 menyediakan dua sistem layout modern untuk membantu proses penataan elemen pada halaman website, yaitu Flexbox dan Grid. Keduanya dirancang untuk menggantikan teknik lama seperti float, table, dan position.

1. Flexbox (Flexible Box Layout)

Flexbox adalah metode layout yang sangat fleksibel dan dibuat untuk mengatur tata letak elemen baik secara horizontal maupun vertikal dalam satu dimensi.

Istilah Penting Flexbox

IstilahPenjelasan
Flex ContainerElemen pembungkus yang diberi property display: flex
Flex ItemsSemua elemen di dalam container
Main AxisArah utama (bisa horizontal atau vertikal tergantung setting)
Cross AxisArah silang dari main axis
justify-contentMengatur posisi item secara horizontal (main axis)
align-itemsMengatur posisi item secara vertikal (cross axis)

Contoh Flexbox Dasar

<div class="flex-container">
  <div class="item">A</div>
  <div class="item">B</div>
  <div class="item">C</div>
</div>

.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
  background: #eef1ff;
  padding: 20px;
}

.item {
  background: #4e73df;
  color: white;
  padding: 15px;
  border-radius: 8px;
}

2. CSS Grid Layout

Grid merupakan metode layout dua dimensi yang dapat mengatur elemen baik dalam bentuk baris (rows) maupun kolom (columns). Grid sangat cocok untuk membangun layout halaman website yang kompleks.

Istilah Penting CSS Grid

IstilahPenjelasan
Grid ContainerElemen utama yang diberi display: grid
Grid ItemsElemen-elemen yang berada di dalam grid container
grid-template-columnsMendefinisikan jumlah kolom serta ukurannya
grid-template-rowsMendefinisikan jumlah baris serta ukurannya
gapJarak antar item dalam grid

Contoh CSS Grid Dasar

<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>

.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 10px;
  background: #e7f7e7;
  padding: 20px;
}

.item {
  background: #22c55e;
  color: white;
  padding: 15px;
  border-radius: 8px;
}

3. Perbandingan Flexbox vs Grid

PerbandinganFlexboxGrid
Dimensi1 Dimensi2 Dimensi
Arah SusunanHanya baris atau kolomBaris dan kolom
KegunaanKomponen kecil / navbarLayout halaman
ContohMenu, Card, ButtonDashboard, Gallery

Kesimpulan: Gunakan Flexbox jika mengatur komponen satu arah seperti menu atau tombol, dan gunakan Grid untuk layout dua dimensi seperti tampilan dashboard atau galeri foto.
