Teori Animasi dalam Pengembangan Website

Animasi pada website adalah pergerakan atau perubahan visual yang dibuat untuk meningkatkan pengalaman pengguna (UI/UX), memperjelas informasi, atau membuat tampilan web lebih menarik.

1. Animasi sebagai Visual Feedback

Animasi membantu memberi tahu pengguna bahwa suatu aksi berhasil atau sedang berlangsung.

Contoh:

<button class="btn">Klik Saya</button> <style> .btn { padding: 10px 20px; background: #3498db; color: #fff; border: none; transition: 0.2s; } .btn:active { transform: scale(0.9); } </style>

2. Animasi sebagai Guiding Attention

Animasi membantu mengarahkan perhatian pengguna ke elemen penting.

Contoh:

<p class="fade">Selamat datang di website kami!</p> <style> .fade { opacity: 0; animation: fadeIn 2s forwards; } @keyframes fadeIn { to { opacity: 1; } } </style>

3. Animasi sebagai Navigation Aid

Animasi memberikan transisi halus antar bagian website.

Contoh:

<div class="menu">Menu Samping</div> <style> .menu { width: 200px; height: 100vh; background: #2ecc71; position: fixed; left: -200px; top: 0; transition: 0.3s; } .menu:hover { left: 0; } </style>
